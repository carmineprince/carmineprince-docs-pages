移动端纯CSS实现隐藏滚动条但仍具有滚动的效果

移动端页面只要兼容 Chrome 和 Safari 就够了，所以可以使用自定义滚动条的伪类选择器 ::-webkit-scrollbar 来隐藏滚动条。

  #kind-page-sidebar::-webkit-scrollbar {
    display: none;
  }
