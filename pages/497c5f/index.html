Flask项目构建基础

# Flask项目构建基础

# 一、安装pipenv工具（全局已安装跳过即可）

pip install pipenv
1

# 二、创建项目目录并cd到项目目录

# 三、创建虚拟环境

pipenv install
1

# 四、进入虚拟环境

pipenv shell
1

# 五、退出虚拟环境

exit
1

# 六、在虚拟环境中安装flask

pipenv install flask
1

# 七、在虚拟环境中检查项目依赖

pipenv graph
1

# 八、查找虚拟环境目录地址

pipenv --venv
1
